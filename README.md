# Introduction

CTF（Capture The Flag）中文一般译作夺旗赛，在网络安全领域中指的是网络安全技术人员之间进行技术竞技的一种比赛形式。

本书内容包含CTF解题需要的基础知识和实践，帮助读者在CTF实践中学习网络安全技术。

正如书名所言，学完此书并练习后，不但没有精通，反而会觉得安全技术没有入门。

本书在讲解知识点时涉及到的，同时也是推荐读者学习CTF的优秀练习靶场包含：
